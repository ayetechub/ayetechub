## 🎨 Replacing Logos

Open `index.html` and find any `<div class="logo-box">AYE</div>` element. Replace with:

```html
<img src="assets/images/logo.png" alt="AYE Tech Hub" style="width:42px;height:42px;object-fit:contain;" />
```

The HTML has comments showing exactly where each logo goes.
